## в—Ҳ What Is This?

**Virus Render** is an educational visualisation that draws a structurally accurate model of a **SARS-CoV-2 (coronavirus)** particle вҖ” layer by layer, in real time вҖ” using nothing but Python's built-in `turtle` graphics library and `math`.

Every layer corresponds to a real biological structure. Every radius is proportionally scaled. The animation draws from the **outside inward**, mirroring how electron microscopy reveals viral anatomy.

## в—Ҳ Biological Accuracy

The render depicts **six distinct anatomical layers** of a real coronavirus, drawn in order from outermost to innermost:

| Layer | Structure | Biological Role |
|:-----:|-----------|-----------------|
| `1` | **Lipid Bilayer Envelope** | Outer membrane derived from the host cell |
| `2` | **Spike Proteins (S)** | 28 trimeric spikes вҖ” the "crown" that gives coronaviruses their name; primary vaccine target |
| `3` | **Matrix Proteins (M)** | Most abundant structural protein; anchors the envelope |
| `4` | **Capsid Shell** | Icosahedral protein coat protecting the genome |
| `5` | **Nucleocapsid Core** | Dense protein-RNA complex |
| `6` | **RNA Genome** | Single-stranded +RNA; drawn as a golden-angle spiral (~30 kb genome) |

## в—Ҳ Features

- рҹ§¬ &nbsp; **Biologically layered** вҖ” six distinct structures, each anatomically distinct
- рҹҺ¬ &nbsp; **Animated drawing** вҖ” `tracer(1, 18)` renders each line stroke visibly in real time
- рҹ”¬ &nbsp; **Proportionally scaled** вҖ” radii are relative to each other, not arbitrary
- рҹҢҖ &nbsp; **Golden-angle RNA spiral** вҖ” 37В° rotation per step approximates natural nucleic acid packing
- рҹ“җ &nbsp; **Icosahedral capsid** вҖ” alternating inner/outer vertices simulate triangulated facets
- рҹ–Ө &nbsp; **Neon-on-black aesthetic** вҖ” high-contrast palette inspired by bio-imaging software
- рҹ“Ұ &nbsp; **Zero dependencies** вҖ” only Python stdlib (`turtle`, `math`)

## в—Ҳ Animation Speed Tuning

Speed is controlled per-pen. Change the `speed=` argument in each `make_pen()` call:

| `speed` value | Effect |
|:---:|---|
| `1` | Very slow вҖ” meditative, good for demonstrations |
| `4вҖ“5` | Slow and deliberate вҖ” default for envelope & spikes |
| `6вҖ“8` | Medium вҖ” capsid facets, core |
| `10` | Fast вҖ” used for RNA (400 steps) |
| `0` | **Instant** вҖ” bypasses animation entirely |

Global step delay is set via:
```python
screen.tracer(1, 18)   # (steps_per_update, delay_in_ms)
```
Increase `18` вҶ’ slower. Decrease вҶ’ faster.
